# Supermarket Sales EDA

## Overview

This project performs Exploratory Data Analysis (EDA) on supermarket sales data to identify sales patterns, customer behavior, and product performance using Python.

## Tools Used

Python

Pandas

NumPy

Matplotlib

Seaborn

## Analysis Performed

Data cleaning and preprocessing

Sales trend analysis

Product category performance

Customer purchasing patterns

Visual data analysis using charts and graphs

## Key Insights

Identified top-performing product categories

Analyzed sales distribution across branches

Observed customer purchasing behavior and payment trends

## Conclusion

The analysis helped extract meaningful business insights from the sales data and demonstrated how data analytics can support better business decision-making.
